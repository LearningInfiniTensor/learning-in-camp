# learning in camp

Opencamp 训练营通用学习工具。

## 环境准备

- cargo；
- git；

## 功能

```shell
cargo xtask help
```

```plaintext
Usage: xtask <COMMAND>

Commands:
  setup  安装指定开发环境
  learn  配置指定课程仓库
  eval   评分
  help   Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
